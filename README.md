#KibotoRecipes

Kiboto setup is a bit complicated, so this repo is a collection of useful patterns and setup scripts.

## Dependencies
```
aptitude
python 2.7, 3.2, 3.3, or 3.4
pip
git
vim
wget
nodejs
```

## Install
This one's easy:

```bash
./install
```

## Local Demo
Run a local demo, using the sample game and bots provided in the repos:
```bash
./start_local_demo
```
